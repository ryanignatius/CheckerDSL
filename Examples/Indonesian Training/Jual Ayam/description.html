Jual Ayam

Batas Waktu 1 detik
Batas Memori 32 MB

Deskripsi

Ayam-ayam Pak Blangkon sudah gemuk, saatnya dijual. Pak Blangkon segera membawa N (1 <= N <= 100000) ayam dari peternakannya untuk mengunjungi N rumah penduduk. Pak Blangkon berencana untuk menjual satu ayam di setiap rumah yang ia kunjungi. Sebagai informasi, letak peternakan dan rumah-rumah penduduk membentang membentuk sebuah garis lurus dengan peternakan Pak Blangkon berada di paling Timur.

Harga ayam di setiap rumah berbeda-beda. Rumah ke-j bersedia membeli ayam seharga P_j (1 <= P_j <= 1000000) per gram. Jarak antara peternakan Pak Blangkon dengan rumah ke-j adalah D_j (1 <= D_j <= 1000000) meter. Setiap ayam memiliki berat yang berbeda. Ayam ke-i memiliki berat W_i (1 <= W_i <= 1000000) gram. Membawa satu gram ayam menempuh jarak sejauh satu meter memerlukan biaya tambahan T (1 <= T <= 100) sebagai biaya bahan bakar.

Bantu Pak Blangkon untuk menentukan di rumah mana setiap ayam harus di jual agar total keuntungan yang diperoleh semaksimal mungkin. Perhatikan bahwa, Pak Blangkon bisa saja pasti rugi, jika demikian, minimalkan nilai kerugiannya (Pak Blangkon harus tetap menjual N ayam tersebut).

Format Masukan

Baris-1: dua bilangan bulat N (1 <= N <= 100000) dan T (1 <= T <= 100)
Baris-2: N buah bilangan bulat, di mana bilangan bulat ke-i menyatakan berat ayam ke-i, W_i (1 <= W_i <= 1000000)
Baris-3: N buah bilangan bulat, di mana bilangan bulat ke-j menyatakan jarak rumah ke-j dari peternakan, D_j (1 <= D_j <= 1000000)
Baris-4: N buah bilangan bulat, di mana bilangan bulat ke-j menyatakan harga dibayar rumah ke-j per gram ayam, P_i (1 <= P_i <= 1000000)

Format Keluaran

Baris-1: N buah bilangan bulat, di mana bilangan bulat ke-j menyatakan ayam yang mana yang harus dijual di rumah ke-j.

Contoh Masukan

3 1
10 20 15
